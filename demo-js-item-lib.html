About Catalog - list of element

Catalog - list of elements is a prototype for an online film catalog, book of orders, etc. It is a cross browser Spa application and has the corresponding advantages. It allows a visitor to have the following conveniences:

  • to select "Favorite"
  • to rate items
  • to filter Favorite or All
  • to look N-th items in a page at once
  • to list the pages forth and back
  • to see all item features
  • to edit item features
  • to save item features
  • to come back to the item list

This application provides basic UI and UX.
